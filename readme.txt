Adds support for the "WP Subtitle" plugin to the "Better Internal Link Search" plugin.

## Description

Better Internal Link Search improves the default internal link searching feature in a number of ways. What's missing is the support for searching subtitles which are added by the WP Subtitle plugin.

### Features

* Searches subtitles added by WP Subtitle in the internal link search popup.

Installing WP Subtitle Support for Better Internal Link Search is just like installing most other plugins. [Check out the codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins) if you have any questions.
